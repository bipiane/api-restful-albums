# API RESTFul Albums

API para la gestión de álbumes con imagenes. 
Una excusa para poner en practica las siguientes tecnologías y más! ;)

- Node.js
- Express.js
- MongoDB
- Mocha Test
- Postman Collection 

### Setup
```sh
  $ npm install && npm run start  
```

### Test
```sh
  $ npm run test  
```

### Tools
- [Robomongo](https://robomongo.org)
- [Postman](https://www.getpostman.com)
